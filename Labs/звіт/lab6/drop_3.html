Установка WEB-сервера

Завантажимо Open Server - це набір програмного забезпечення, який включає Apache HTTP Server, PHP 8, різноманітні СУБД та інші веб-сервери.

Рис. 1 - процес установки

Вибираємо Apache HTTP Server для PHP 8 та сам PHP 8. Інше можна вибрати за бажанням

Рис. 2 - продовження процесу установки

Open Server успішно встановлено. Прописавши змінну оточення PATH для PHP можна продовжувати.
